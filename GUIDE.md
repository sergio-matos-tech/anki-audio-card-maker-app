# Simple User Guide

This guide shows how to use Anki Stream Aligner.

## 1. Prepare Your Lesson Folder

Put the lesson document and audio files in the same folder.

Example:

```text
DRINK UP - Full Lesson.pdf
1 - track.mp3
2 - track.mp3
3 - track.mp3
```

The folder must have:

- One `.pdf` or `.txt` lesson file
- Numbered `.mp3` files
- Audio numbers in the same order as the sentences

## 2. Open The App

Run:

```text
AnkiStreamAligner.exe
```

If you are running from source, use:

```powershell
python -m src.ui
```

## 3. Choose Your Anki Profile

Keep `User 1` if that is your Anki profile name.

If your Anki profile has another name, type it exactly as it appears in Anki.

## 4. Select The Lesson Folder

Click `Select Source Folder`.

Choose the folder that contains the lesson document and the numbered MP3 files.

## 5. Generate The Import File

Click `GENERATE & SYNC ASSETS`.

Choose where to save the generated `.txt` file.

The app will:

- Read the lesson document
- Create the Anki import file
- Copy the used MP3 files into Anki's media folder

## 6. Import Into Anki

Open Anki and click `Import File`.

Select the `.txt` file generated by the app.

Use these import settings:

| Setting | Value |
| --- | --- |
| Note type | Basic, or your custom note type |
| Deck | Any deck you want |
| Field separator | Tab |

Check the field mapping:

| Input field | Anki field |
| --- | --- |
| Field 1 | Front |
| Field 2 | Back |

Click `Import`.

## Done

Your cards should now show the English sentence on the front, the Portuguese translation on the back, and the audio should play from Anki.
