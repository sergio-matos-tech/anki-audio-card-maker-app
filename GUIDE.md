# Anki Audio Card Maker Guide

Anki Audio Card Maker is a beta app for Windows. It creates Anki import files from bilingual lessons and numbered audio files.

## 1. Prepare Your Lesson Folder

Put the lesson document and audio files in the same folder.

Example:

```text
DRINK UP - Full Lesson.pdf
1 - track.mp3
2 - track.mp3
3 - track.mp3
```

You need:

- One `.pdf` or `.txt` lesson file
- Numbered `.mp3` files
- Audio files in the same order as the lesson sentences

## 2. Open The App

Run:

```text
AnkiAudioCardMaker.exe
```

## 3. Choose Your Anki Profile

Keep `User 1` if that is your Anki profile name.

If your profile has another name, type it exactly as it appears in Anki.

## 4. Choose The Lesson Folder

Click `Choose lesson folder`.

Select the folder with the lesson document and numbered MP3 files.

## 5. Create The Import File

Click `Create Anki import file`.

Choose where to save the generated `.txt` file.

The app will read the lesson, create the import file, and copy the used MP3 files into Anki's media folder.

## 6. Import Into Anki

Open Anki and click `Import File`.

Select the `.txt` file generated by the app.

Use these settings:

| Setting | Value |
| --- | --- |
| Note type | Basic, or your custom note type |
| Deck | Any deck you want |
| Field separator | Tab |

Check the field mapping:

| Input field | Anki field |
| --- | --- |
| Field 1 | Front |
| Field 2 | Back |

Click `Import`.

## Done

Your cards should show the English sentence on the front, the Portuguese translation on the back, and the audio should play in Anki.
